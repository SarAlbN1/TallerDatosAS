Arquitectura de Dos Niveles

Nivel de Presentación (MPA)

Esta aplicación web multi-página que consume servicios del nivel de datos

  • HTML, CSS, JavaScript vanilla
  • Múltiples páginas (MPA)
  • Consumo de servicios REST y SOAP
↓

Nivel de Datos

Servicios que exponen la lógica de negocio y acceso a datos

  • Servicios REST: /api/products, /api/organizations, /api/categories
  • Servicios SOAP: /ws con operaciones GetProducts y CreateProduct
  • Base de datos MySQL con JPA

Características del Sistema

Base de Datos con Docker

MySQL 8.0 desplegado con Docker Compose, con persistencia de datos

Cliente Pesado Java

Spring Boot con JPA para consultas a la base de datos

Aplicación Web SPA

React SPA moderna para gestión de productos
